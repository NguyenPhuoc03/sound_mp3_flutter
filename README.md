# 🎵 Sound MP3 Flutter

Đây là frontend cho ứng dụng nghe nhạc **Sound MP3**.

## ⚙️ Công nghệ sử dụng

- Flutter
- State Management: Provider
- Networking libraries: http

## 🌐 Backend API

Backend **Sound MP3** được phát triển với Nodejs, ExpressJS.
